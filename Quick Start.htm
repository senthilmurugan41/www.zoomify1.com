Zoomify Free Quick Start

Zoomify Free delivers simple zoom-and-pan

Make any image zoomable, no matter how large! Zoomify Free includes everything you need to start - drag-drop Converter, simple image Viewer, and template web page.

And Zoomify Free is completely compatible with all Zoomify products. Upgrade at any time for features like fullscreen viewing, complete Toolbar options, bookmarking, and slideshows - no need to reconvert your images!

 

Just 3 steps to zoom!

1. Run the Zoomify Free Converter and select your image file to create a Zoomify Image folder.

2. Edit TemplateWebPage.htm and change the value 'ZoomifyImageExample' to your image folder's name.

3. Copy your image folder and the files TemplateWebPage.htm and ZoomifyImageViewerFree-min.js to your webserver.

 

OR easily add Zoomify Free to your own web pages with just three lines:

1. The first line tells the web page to include the Viewer script file - in the <head> of the page:
      <script type="text/javascript" src="ZoomifyImageViewerFree-min.js"></script>

2. The second line is also in the <head> of the page. It creates the Viewer and tells it where in the page to display:
      <script type="text/javascript"> Z.showImage("myContainer", "ZoomifyImageExample"); </script>

3. The third line creates the container element for the Viewer to exist in: "myContainer" in the <body> of the page:
      <div id="myContainer" style="width:900px; height:500px; margin:auto; border:1px; border-style:solid; " ></div>

See the Zoomify website demos and product examples for additional options.
